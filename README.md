# My First Slack Bot

A Slack bot built with Node.js and Slack Bolt. When I was making this, I initially struggled with figuring out how the code resulted in these commands as I did not know javascript before and had to learn before this project. Because of the new language, it took me a while to begin the process, not to mention, this was also my first time using vs code, and I had to figure that out as well.

Commands:
- /sb09-hello : says pong and returns the latency
- /sb09-catfact : gives a random cat fact
- /sb09-joke : tells a random joke
