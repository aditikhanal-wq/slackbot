# My First Slack Bot

A Slack bot built with Node.js and Slack Bolt. When I was making this, I initially struggled with figuring out how the code resulted in these commands as I did not know javascript before and had to learn before this project. Because of the new language, it took me a while to begin the process, not to mention, this was also my first time using vs code, and I had to figure that out as well. This project took me a few days to complete. I especially struggled with the cat fact and the joke as accessing the internet to do so kept failing for me. I have 3 commands that are listed below, along with what they do. I also struggled a lot with hosting the process on Nest, as I did that on a separate day than actually coding the project. When I came back to do so, whenever I tried to do anything, I was just met with a lot of errors, so I had to do a lot of trouble shooting. Eventually, I was successful, and was able to make the fully functioning slack bot.

Commands:
- /sb09-hello : says pong and returns the latency
- /sb09-catfact : gives a random cat fact
- /sb09-joke : tells a random joke
